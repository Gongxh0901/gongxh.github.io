《creator-inspector》🏆 Cocos官方推荐

即插即用的 Cocos Creator 运行时调试与内存分析工具：左侧实时节点树（定位高亮、属性即改即生效），右侧资源内存占用盘点（按 Bundle/类型/路径分组与筛选）。兼容 Creator 2.x/3.x，零依赖，开发期一键启用。项目由佳航开发，获得 Cocos 官方推荐。

《游戏框架》💼 商业项目实战

基于 Cocos Creator 的一站式游戏开发框架，经过多个商业项目实战验证。集成了 FairyGUI 界面系统、 多平台小游戏 API 封装、资源管理、热更新、平台适配等模块。支持 TypeScript 完整类型定义， 提供开箱即用的 UI 装饰器、窗口管理、计时器、平台工具等功能。框架已发布到 npm， 配套完整文档和 Demo 项目，帮助开发者快速构建高质量的游戏产品。
